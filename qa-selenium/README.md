## Creación de tests automatizados con Selenium utilizando el patrón POM (PageObject Model)

### Contexto

- Navegar a la URL: [http://www.automationpractice.pl/index.php](https://parabank.parasoft.com/parabank/index.htm)
- Navegar a la página de login.
- Hacer login (la cuál nos lleva a la página "My account").
- Navegar de nuevo a la Home page.
- Seleccionar el primer producto.
- Añadir el producto a la Shopping cart.
- Proceder al checkout.
- Verificar que hay un producto añadido en la Shopping cart.
